Modern, minimal e-commerce WordPress theme designed for Elementor with WooCommerce and WPML support.

== Description ==

Luwi Elementor is a premium, performance-focused WordPress theme built for modern e-commerce stores. Designed to work seamlessly with Elementor page builder, WooCommerce, and WPML, it provides a clean foundation for beautiful online stores.

= Key Features =

* **Elementor Theme Builder** — Full support for custom headers, footers, product templates, and archive layouts
* **WooCommerce Optimized** — Beautiful product cards, galleries, cart, and checkout styling out of the box
* **Light & Dark Mode** — Built-in color mode toggle with system preference detection
* **Accessibility Ready** — WCAG 2.1 AA compliant with high contrast mode, reduced motion, and font size scaling
* **WPML Compatible** — Translation-ready with full RTL support
* **Performance First** — No jQuery dependency, minimal CSS/JS, preloaded fonts
* **Customizer Options** — Brand colors, typography, layout, and accessibility settings
* **LuwiPress Integration** — Enhanced AI-powered features when paired with LuwiPress plugin

= Recommended Plugins =

* Elementor (required for full Theme Builder functionality)
* WooCommerce (for e-commerce features)
* LuwiPress (for AI content generation, SEO automation, and smart widgets)
* WPML or Polylang (for multilingual stores)

== Installation ==

1. Upload the `luwi-emerald` folder to `/wp-content/themes/`
2. Activate the theme through the WordPress admin panel
3. Install and activate Elementor (if not already installed)
4. Go to **Appearance > Customize > Luwi Theme Options** to configure your settings
5. Use Elementor Theme Builder to create custom headers, footers, and product templates
